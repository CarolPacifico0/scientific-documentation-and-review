\## Objective Produce a baseline characteristics table for the safety
population.

\## Input Data - Subject-level dataset - Demographics and baseline
variables

\## Analysis Population - Safety population - Subjects who received at
least one dose

\## Variables Included - Age - Sex - Treatment group

\## Statistical Approach - Continuous variables summarized using mean,
SD, median - Categorical variables summarized using counts and
percentages

\## Assumptions - One record per subject - Baseline variables measured
prior to treatment

\## Outputs - Baseline characteristics table (Table 1)
